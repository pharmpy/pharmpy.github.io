Datasets in Pharmpy¶

Datasets in Pharmpy are represented using the pharmpy.data.PharmDataFrame class. It is a subclass of the pandas DataFrame and have some additions specific to Pharmacometrics.

\[\]

Retrieving the dataset from a model¶

from pharmpy import Model

model = Model(path / "pheno_real.mod")
df = model.dataset

A new or updated dataset can be set to a model

import numpy as np

df['DV'] = np.log(df['DV'], where=(df['DV'] != 0))
model.dataset = df
