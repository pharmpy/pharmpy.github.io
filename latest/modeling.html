Modeling¶

While the pharmpy.model.Model class can be directly manipulated with low level operations the modeling module offers higher level operations and transformations for building a model. These transformations are also available via the Pharmpy command line interface.

PK models and ODE systems¶

from pharmpy import Model

model = Model(path / "pheno_real.mod")

The ODE system of a PK model can be converted from having a compartmental description to be described with an explicit ODE-system.

from pharmpy.modeling import explicit_odes

print(model.statements.ode_system)
explicit_odes(model)
print(model.statements.ode_system)

┌───────┐       ┌──────┐
│CENTRAL│──CL/V→│OUTPUT│
└───────┘       └──────┘

⎧d                  -CL⋅A_CENTRAL(t) 
⎪──(A_CENTRAL(t)) = ─────────────────
⎪dt                         V        
⎨d                 CL⋅A_CENTRAL(t)
⎪──(A_OUTPUT(t)) = ───────────────
⎪dt                       V       
⎪A_CENTRAL(0) = AMT
⎩A_OUTPUT(0) = 0

For NONMEM models this means going from any of the compartmental ADVANS (ADVAN1-4, ADVAN10-12) to coding using an explicit $DES.

model.update_source()
print(model)

$PROBLEM PHENOBARB SIMPLE MODEL
$DATA 'pheno.dta' IGNORE=@
$INPUT ID TIME AMT WGT APGR DV FA1 FA2
$SUBROUTINE ADVAN6

$MODEL TOL=3 COMPARTMENT=(CENTRAL DEFDOSE)
$PK
IF(AMT.GT.0) BTIME=TIME
TAD=TIME-BTIME
TVCL=THETA(1)*WGT
TVV=THETA(2)*WGT
IF(APGR.LT.5) TVV=TVV*(1+THETA(3))
CL=TVCL*EXP(ETA(1))
V=TVV*EXP(ETA(2))
S1=V

$DES
DADT(1) = -A(1)*CL/V
$ERROR
W=F
Y=F+W*EPS(1)
IPRED=F
IRES=DV-IPRED
IWRES=IRES/W

$THETA (0,0.00469307) ; PTVCL
$THETA (0,1.00916) ; PTVV
$THETA (-.99,.1)
$OMEGA DIAGONAL(2)
 0.0309626  ;       IVCL
 0.031128  ;        IVV

$SIGMA 1e-7
$ESTIMATION METHOD=1 INTERACTION
$COVARIANCE UNCONDITIONAL
$TABLE ID TIME DV AMT WGT APGR IPRED PRED RES TAD CWRES NPDE NOAPPEND
       NOPRINT ONEHEADER FILE=sdtab1

Adding covariate effects¶

model = Model(path / "pheno_real.mod")

Covariate effects may also be applied to a model.

from pharmpy.modeling import add_covariate_effect
add_covariate_effect(model, 'CL', 'WGT', 'lin')

Here, CL indicates the name of the parameter onto which you want to apply the effect, WGT is the covariate, and lin (linear function on continuous covariates) is the effect you want to apply. See pharmpy.modeling.add_covariate_effect for effects with available templates.

model.update_source()
print(model)

$PROBLEM PHENOBARB SIMPLE MODEL
$DATA 'pheno.dta' IGNORE=@
$INPUT ID TIME AMT WGT APGR DV FA1 FA2
$SUBROUTINE ADVAN1 TRANS2

$PK
IF(AMT.GT.0) BTIME=TIME
TAD=TIME-BTIME
TVCL=THETA(1)*WGT
TVV=THETA(2)*WGT
IF(APGR.LT.5) TVV=TVV*(1+THETA(3))
CL=TVCL*EXP(ETA(1))
WGT_MEDIAN = 1.30000
CLWGT = (WGT - WGT_MEDIAN)*THETA(4) + 1
CL = CLWGT*TVCL*EXP(ETA(1))
V=TVV*EXP(ETA(2))
S1=V

$ERROR
W=F
Y=F+W*EPS(1)
IPRED=F
IRES=DV-IPRED
IWRES=IRES/W

$THETA (0,0.00469307) ; PTVCL
$THETA (0,1.00916) ; PTVV
$THETA (-.99,.1)
$THETA  (-100000,0.001,100000) ; COVEFF1
$OMEGA DIAGONAL(2)
 0.0309626  ;       IVCL
 0.031128  ;        IVV

$SIGMA 1e-7
$ESTIMATION METHOD=1 INTERACTION
$COVARIANCE UNCONDITIONAL
$TABLE ID TIME DV AMT WGT APGR IPRED PRED RES TAD CWRES NPDE NOAPPEND
       NOPRINT ONEHEADER FILE=sdtab1

Pharmpy also supports user formatted covariate effects.

model = Model(path / "pheno_real.mod")
user_effect = 'median - cov + theta'
add_covariate_effect(model, 'CL', 'WGT', user_effect)

It is necessary that the names follow the same format as in user_effect, meaning that the covariate is denoted as cov, the theta as theta (or, if multiple thetas: theta1, theta2 etc.), and the mean or median as mean and median, respectively. This is in order for the names to be substituted with the correct values.

model.update_source()
print(model)

$PROBLEM PHENOBARB SIMPLE MODEL
$DATA 'pheno.dta' IGNORE=@
$INPUT ID TIME AMT WGT APGR DV FA1 FA2
$SUBROUTINE ADVAN1 TRANS2

$PK
IF(AMT.GT.0) BTIME=TIME
TAD=TIME-BTIME
TVCL=THETA(1)*WGT
TVV=THETA(2)*WGT
IF(APGR.LT.5) TVV=TVV*(1+THETA(3))
CL=TVCL*EXP(ETA(1))
WGT_MEDIAN = 1.30000
CLWGT = -WGT + WGT_MEDIAN + THETA(4)
CL = CLWGT*TVCL*EXP(ETA(1))
V=TVV*EXP(ETA(2))
S1=V

$ERROR
W=F
Y=F+W*EPS(1)
IPRED=F
IRES=DV-IPRED
IWRES=IRES/W

$THETA (0,0.00469307) ; PTVCL
$THETA (0,1.00916) ; PTVV
$THETA (-.99,.1)
$THETA  (-100000,0.001,100000) ; COVEFF1
$OMEGA DIAGONAL(2)

 0.031128  ;        IVV

$SIGMA 1e-7
$ESTIMATION METHOD=1 INTERACTION
$COVARIANCE UNCONDITIONAL
$TABLE ID TIME DV AMT WGT APGR IPRED PRED RES TAD CWRES NPDE NOAPPEND

Transformation of etas¶

Boxcox¶

model = Model(path / "pheno_real.mod")

To apply a boxcox transformation, input a list of the etas of interest.

from pharmpy.modeling import boxcox
boxcox(model, ['ETA(1)'])
model.update_source()
print(model)

$PROBLEM PHENOBARB SIMPLE MODEL
$DATA 'pheno.dta' IGNORE=@
$INPUT ID TIME AMT WGT APGR DV FA1 FA2
$SUBROUTINE ADVAN1 TRANS2

$PK
ETAB1 = (EXP(ETA(1))**THETA(4) - 1)/THETA(4)
IF(AMT.GT.0) BTIME=TIME
TAD=TIME-BTIME
TVCL=THETA(1)*WGT
TVV=THETA(2)*WGT
IF(APGR.LT.5) TVV=TVV*(1+THETA(3))
CL = TVCL*EXP(ETAB1)
V=TVV*EXP(ETA(2))
S1=V

$ERROR
W=F
Y=F+W*EPS(1)
IPRED=F
IRES=DV-IPRED
IWRES=IRES/W

$THETA (0,0.00469307) ; PTVCL
$THETA (0,1.00916) ; PTVV
$THETA (-.99,.1)
$THETA  (-3,0.01,3) ; lambda1
$OMEGA DIAGONAL(2)
 0.0309626  ;       IVCL
 0.031128  ;        IVV

$SIGMA 1e-7
$ESTIMATION METHOD=1 INTERACTION
$COVARIANCE UNCONDITIONAL
$TABLE ID TIME DV AMT WGT APGR IPRED PRED RES TAD CWRES NPDE NOAPPEND
       NOPRINT ONEHEADER FILE=sdtab1

This can be done for one or multiple etas. The new model will have new statements where ETAB1 is a boxcox transformation of ETA(1).

If no list is provided, all etas will be updated.

model = Model(path / "pheno_real.mod")
boxcox(model)
model.update_source()
print(model)

$PROBLEM PHENOBARB SIMPLE MODEL
$DATA 'pheno.dta' IGNORE=@
$INPUT ID TIME AMT WGT APGR DV FA1 FA2
$SUBROUTINE ADVAN1 TRANS2

$PK
ETAB1 = (EXP(ETA(1))**THETA(4) - 1)/THETA(4)
ETAB2 = (EXP(ETA(2))**THETA(5) - 1)/THETA(5)
IF(AMT.GT.0) BTIME=TIME
TAD=TIME-BTIME
TVCL=THETA(1)*WGT
TVV=THETA(2)*WGT
IF(APGR.LT.5) TVV=TVV*(1+THETA(3))
CL = TVCL*EXP(ETAB1)
V = TVV*EXP(ETAB2)
S1=V

$ERROR
W=F
Y=F+W*EPS(1)
IPRED=F
IRES=DV-IPRED
IWRES=IRES/W

$THETA (0,0.00469307) ; PTVCL
$THETA (0,1.00916) ; PTVV
$THETA (-.99,.1)
$THETA  (-3,0.01,3) ; lambda1
$THETA  (-3,0.01,3) ; lambda2
$OMEGA DIAGONAL(2)
 0.0309626  ;       IVCL
 0.031128  ;        IVV

$SIGMA 1e-7
$ESTIMATION METHOD=1 INTERACTION
$COVARIANCE UNCONDITIONAL
$TABLE ID TIME DV AMT WGT APGR IPRED PRED RES TAD CWRES NPDE NOAPPEND
       NOPRINT ONEHEADER FILE=sdtab1

Approximate t-distribution¶

Applying an approximate t-distribution transformation of etas is analogous to a boxcox transformation. The input is similarly a list of etas, and if no list is provided all etas will be transformed.

model = Model(path / "pheno_real.mod")
from pharmpy.modeling import tdist
tdist(model, ['ETA(1)'])
model.update_source()
print(model)

$PROBLEM PHENOBARB SIMPLE MODEL
$DATA 'pheno.dta' IGNORE=@
$INPUT ID TIME AMT WGT APGR DV FA1 FA2
$SUBROUTINE ADVAN1 TRANS2

$PK
ETAT1 = ((ETA(1)**2 + 1)/(4*THETA(4)) + (5*ETA(1)**4 + 16*ETA(1)**2 + 3)/(96*THETA(4)**2) + (3*ETA(1)**6 + 19*ETA(1)**4 + 17*ETA(1)**2 - 15)/(384*THETA(4)**3) + 1)*ETA(1)
IF(AMT.GT.0) BTIME=TIME
TAD=TIME-BTIME
TVCL=THETA(1)*WGT
TVV=THETA(2)*WGT
IF(APGR.LT.5) TVV=TVV*(1+THETA(3))
CL = TVCL*EXP(ETAT1)
V=TVV*EXP(ETA(2))
S1=V

$ERROR
W=F
Y=F+W*EPS(1)
IPRED=F
IRES=DV-IPRED
IWRES=IRES/W

$THETA (0,0.00469307) ; PTVCL
$THETA (0,1.00916) ; PTVV
$THETA (-.99,.1)
$THETA  (3,80,100) ; df1
$OMEGA DIAGONAL(2)
 0.0309626  ;       IVCL
 0.031128  ;        IVV

$SIGMA 1e-7
$ESTIMATION METHOD=1 INTERACTION
$COVARIANCE UNCONDITIONAL
$TABLE ID TIME DV AMT WGT APGR IPRED PRED RES TAD CWRES NPDE NOAPPEND
       NOPRINT ONEHEADER FILE=sdtab1

John Draper¶

Similarly, a John Draper transformation uses a list of etas as input, if no list is provided all etas will be transformed.

model = Model(path / "pheno_real.mod")
from pharmpy.modeling import john_draper
john_draper(model, ['ETA(1)'])
model.update_source()
print(model)

$PROBLEM PHENOBARB SIMPLE MODEL
$DATA 'pheno.dta' IGNORE=@
$INPUT ID TIME AMT WGT APGR DV FA1 FA2
$SUBROUTINE ADVAN1 TRANS2

$PK
ETAD1 = ((ABS(ETA(1)) + 1)**THETA(4) - 1)*ABS(ETA(1))/(ETA(1)*THETA(4))
IF(AMT.GT.0) BTIME=TIME
TAD=TIME-BTIME
TVCL=THETA(1)*WGT
TVV=THETA(2)*WGT
IF(APGR.LT.5) TVV=TVV*(1+THETA(3))
CL = TVCL*EXP(ETAD1)
V=TVV*EXP(ETA(2))
S1=V

$ERROR
W=F
Y=F+W*EPS(1)
IPRED=F
IRES=DV-IPRED
IWRES=IRES/W

$THETA (0,0.00469307) ; PTVCL
$THETA (0,1.00916) ; PTVV
$THETA (-.99,.1)
$THETA  (-3,0.01,3) ; lambda1
$OMEGA DIAGONAL(2)
 0.0309626  ;       IVCL
 0.031128  ;        IVV

$SIGMA 1e-7
$ESTIMATION METHOD=1 INTERACTION
$COVARIANCE UNCONDITIONAL
$TABLE ID TIME DV AMT WGT APGR IPRED PRED RES TAD CWRES NPDE NOAPPEND
