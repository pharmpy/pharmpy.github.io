Multiple Versions of PythonÂ¶

On Ubuntu:

  1. Install dependencies

sudo apt-get update
sudo apt-get install --no-install-recommends make build-essential libssl-dev zlib1g-dev libbz2-dev libreadline-dev libsqlite3-dev wget curl llvm libncurses5-dev xz-utils tk-dev libxml2-dev libxmlsec1-dev libffi-dev liblzma-dev

  1. Clone repository/install pyenv

git clone https://github.com/pyenv/pyenv.git ~/.pyenv

OR

curl https://pyenv.run | bash

  1. Add to .bashrc so that pyenv is loaded automatically

echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bashrc
echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bashrc
echo -e 'if command -v pyenv 1>/dev/null 2>&1; then\n  eval "$(pyenv init -)"\nfi' >> ~/.bashrc

  1. Install python versions of interest

pyenv install 3.6.10
pyenv install 3.7.7
pyenv install 3.8.3

  1. Change to folder where tests are run (in this case pharmpy) and set local python versions

pyenv local 3.6.10 3.7.7 3.8.3

Sources: https://realpython.com/intro-to-pyenv/ https://github.com/pyenv/pyenv
