pharmpy.execute.engine module¶

Execution Engine¶

Job creator for a Model implementation.

The critical, non-agnostic and central unit to inherit (e.g. NONMEM7). That implementation can be multiclassed dynamically for Tool implementations, if mere duck typing doesn’t cut it.

Note

An Engine implementation is expected to implement all methods/attributes.
