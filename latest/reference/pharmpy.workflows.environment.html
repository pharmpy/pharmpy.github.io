pharmpy.workflows.environment module¶

Execution Environment¶

Platform (e.g. Linux) & system (e.g. SLURM) implementation.

The cluster/local or OS etc. to start jobs on.

class pharmpy.workflows.environment.SystemEnvironment(*args, **kwargs)[source]¶

Bases: pharmpy.workflows.environment.ExecutionEnvironment

Manages system (direct subprocess) execution for an engine on some platform.

Needs asyncio because a running event loop is required to monitor child processes. Execution machinery via concurrent.futures interface.

Variables

  • jobs – Submitted jobs.

  • futures – Future promises of jobs, instances of concurrent.futures.Future.

  • pool – Threading pool, instance of ThreadPoolExecutor.
