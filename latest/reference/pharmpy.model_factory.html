pharmpy.model_factory module¶

Factory for creating a pharmpy model from a “native” object representation.

Definitions¶

pharmpy.model_factory.Model(obj, **kwargs)[source]¶

Factory for creating a Model object from an object representing the model (i.e. path).

Parameters

obj – Currently a path-like object pointing to the model file.

Generic Model if path is None, otherwise appropriate implementation is invoked (e.g. NONMEM7 Model).
