class pharmpy.RandomVariable(name, level, sympy_rv=None)[source]¬∂

Bases: object

A single random variable

Parameters

  • name (str) ‚Äì Name of the random variable

  • level (str) ‚Äì Name of the variability level. The default levels are IIV, IOV and RUV

  • sympy_rv (sympy.RandomSymbol) ‚Äì RandomSymbol to use for this random variable. See also the normal and joint_normal classmethods.

Examples

>>> import sympy
>>> import sympy.stats
>>> from pharmpy import RandomVariable
>>> name = "ETA(1)"
>>> sd = sympy.sqrt(sympy.Symbol('OMEGA(1,1)'))
>>> rv = RandomVariable(name, "IIV", sympy.stats.Normal(name, 0, sd))
>>> rv
