set_dtbs_error_model#

pharmpy.modeling.set_dtbs_error_model(model, fix_to_log=False)[source]#

Dynamic transform both sides

Parameters

  • model (Model) – Pharmpy model

  • fix_to_log (Boolean) – Set to True to fix lambda and zeta to 0, i.e. emulating log-transformed data

Returns

Model – Reference to the same model

Examples

>>> from pharmpy.modeling import load_example_model, set_dtbs_error_model
>>> model = load_example_model("pheno")
>>> set_dtbs_error_model(model)
